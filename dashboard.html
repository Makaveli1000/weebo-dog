Netlify Features Dashboard

Serverless Functions

Run backend code without managing servers. Ideal for APIs, automation, and more. Deployed as part of your site's codebase.

Edge Functions

Run code at the network edge, closer to your users. Perfect for personalization, authentication, and performance-critical tasks.

Netlify Blobs

Persistent, scalable object storage for any kind of data. Great for user-generated content, session data, and more.

Netlify Image CDN

Automated image optimization and transformation. Serve perfectly sized images to your users without any extra build steps.

Environment Variables

Manage your site's configuration and secrets securely. Access them in your build process and in your serverless functions.

Netlify Forms

Handle form submissions without any server-side code. Just add a `netlify` attribute to your HTML form and you're done.

Creating New Sites

Easily create new sites from a Git repository. Netlify's CI/CD will automatically build and deploy your site on every push.

Netlify DB

A serverless database powered by PostgreSQL. It provides a simple way to store and query data from your Netlify Functions.
